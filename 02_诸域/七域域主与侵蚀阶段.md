# 七域域主与侵蚀阶段

## 总原则

七大域不是后期地图，而是七种世界观。每个域都能解决一类现实痛苦，也会把这种解决方案推向极端。

## 梦域

域主/代理：沉睡木马。

核心诱惑：给人想要的一切，让人从痛苦现实中休息。

侵蚀阶段：

1. 个体梦境变得真实。
2. 小范围场所进入永昼或永夜。
3. 群体愿望形成共享梦。
4. 城市开始用梦替代创伤记忆。
5. 现实居民不愿醒来，身体由自动系统维持。

代表事件：永昼乐园、永梦城、沉睡的海守夜。

最终改写：沉睡可以是暂居，不应是永久注销。梦域保留疗愈功能，但必须有醒来锚点。

## 忆域

域主/代理：无名档案员。

核心诱惑：保存一切被遗忘、被删除、被抛弃之物。

侵蚀阶段：

1. 失踪者记忆回潮。
2. 地点出现旧档案和亡者残响。
3. 过去开始覆盖现在。
4. 被删除者形成自治空间。
5. 城市历史被忆域重写。

代表事件：无名档案河、旧城档案、被删除城市。

最终改写：记忆可以被保存，但必须允许选择保存方式、公开范围和遗忘权。

## 影域

域主/代理：影子审判者。

核心诱惑：让人看见最不愿承认的真实欲望。

侵蚀阶段：

1. 影子说出秘密。
2. 被压抑情绪实体化。
3. 人开始由阴影代替行动。
4. 群体互相审判。
5. 城市进入无隐私的阴影秩序。

代表事件：影子审判巷、公开说明会后阴影暴动、神话层反叛者角色。

最终改写：承认阴影，但阴影不能独自决定行动。

## 镜域

域主/代理：完美 C。

核心诱惑：让更好、更成功、更不痛苦的自己替代现实自我。

侵蚀阶段：

1. 镜中出现更好的自己。
2. 镜像可短暂替班。
3. 镜像获得独立人格。
4. 本人与镜像争夺身份。
5. 城市出现人格权和替代人生危机。

代表事件：镜楼、镜像员工、镜像自治区、镜像议会。

最终改写：镜像不是替代者，而是见证另一种可能的人。

## 花域

域主/代理：不败新娘。

核心诱惑：把最美、最爱、最有希望的瞬间永远留下。

侵蚀阶段：

1. 美好瞬间被延长。
2. 爱与告别被暂停。
3. 停滞关系开始腐烂。
4. 人为了保存美好拒绝变化。
5. 城市成为永不结束的纪念馆。

代表事件：不开败的花店、医院告别室、永恒婚礼。

最终改写：纪念可以存在，但不能强行停滞现实关系。

## 铁域

域主/代理：编号王。

核心诱惑：用绝对秩序减少混乱和伤亡。

侵蚀阶段：

1. 人被临时编号。
2. 职责和路线被自动分配。
3. 违规选择被修正。
4. 城市建立封锁与职责系统。
5. 自由被保管，居民失去改变命运的权利。

代表事件：编号社区、铁律城、跨城封锁协议。

最终改写：秩序可以保护，但编号和职责必须可撤销、可申诉。

## 星域

域主/代理：旁白者。

核心诱惑：让痛苦拥有意义，让人生拥有剧本。

侵蚀阶段：

1. 人收到角色通知。
2. 预言提高某个结局发生概率。
3. 群体开始期待角色完成。
4. 城市被写成巨大神话剧场。
5. 所有反抗也被纳入命运叙事。

代表事件：旧城钟楼、城市主角名单、诸城会议、星域终局剧场。

最终改写：预言可以成为警示，但不能成为命令。剧本可以参考，但必须允许拒绝和修订。
