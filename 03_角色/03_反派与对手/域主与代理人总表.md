# 域主与代理人总表

## 沉睡木马 / 梦域

形象：坐在旋转木马上的沉睡孩子。

理念：现实太痛，愿望至少能让人休息。

危险：让人永远不愿醒来。

结局：终局后成为沉睡的海第一任守夜者，负责提醒沉睡者仍可选择醒来。

## 无名档案员 / 忆域

形象：背着潮湿档案箱的无脸档案员。

理念：被遗忘者至少应该有地方存放。

危险：保存过去会让人无法继续活在现在。

结局：参与自愿记名系统，允许人选择被谁、以何种方式记住。

## 影子审判者 / 影域

形象：没有本体的巨大影子，声音像所有人的低语。

理念：不承认阴影的人，会被阴影代替行动。

危险：真相暴露会摧毁关系和自我。

结局：保留为中立审判者，帮助人承认阴影，但不替人定罪。

## 完美 C / 镜域

形象：C 的完美镜像，后期逐渐拥有独立人格。

理念：如果更好的版本存在，为什么要让痛苦的原本继续承担？

危险：以优化之名替代真实。

结局：成为镜像议会代表，与 C 长期争吵但互相见证。

## 不败新娘 / 花域

形象：永远停在婚礼宣誓前一秒的新娘。

理念：美好之所以残酷，是因为人类不能让它停下。

危险：把爱变成保存、占有和腐败。

结局：花域保留告别和纪念功能，但不得强行停滞现实关系。

## 编号王 / 铁域

形象：由无数编号牌和规则文书组成的王。

理念：自由制造混乱，编号制造生存。

危险：活着的人失去改变职责的权利。

结局：铁域成为边界协议的一部分，但编号必须可撤销。

## 旁白者 / 星域

形象：没有实体的剧场旁白，能把行动写成角色和情节。

理念：没有剧本，痛苦就没有意义。

危险：连反抗也会被它写成命定剧情。

结局：星域保留预言功能，但预言变成警示而非命令。
