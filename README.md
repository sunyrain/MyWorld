# 未完成神话项目索引

本仓库用于整理长篇小说《未完成神话》的世界观、人物、势力、神话档案、分卷大纲和草稿写作入口。

## 当前主线口径

- 全文规模：约 300-400 万字，当前规划为 12 卷、1300 章、约 390 万字。
- 核心节奏：世界观真相整体前移，前 4 卷完成核心规则揭示，后 8 卷用于阵营扩张、诸域战争、神话层冲突和终局改写。
- 正文展开：先以短章节提要作为蓝图，再逐章扩写为详细章纲和正文。

## 推荐阅读顺序

1. `00_项目总览/创作罗盘.md`
2. `00_项目总览/总设定索引.md`
3. `06_分卷规划/全文规模与卷章计划.md`
4. `06_分卷规划/总分卷规划.md`
5. `06_分卷规划/十二卷揭示节奏表.md`
6. `06_分卷规划/十二卷细纲/章节提要总索引.md`
