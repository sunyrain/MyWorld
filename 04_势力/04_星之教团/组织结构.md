# 星之教团组织结构

## 表层身份

信仰星、预言、神话秩序和命运角色的秘密组织。

## 核心理念

人类痛苦，是因为他们偏离了自己的命运。只要每个人回到该扮演的角色，世界就会重新完整。

## 组织层级

| 层级 | 功能 |
| --- | --- |
| 真祭司 | 解释星域剧本，决定教团方向 |
| 祭司 | 执行神话显影和角色归位仪式 |
| 祷星者 | 传播命运信仰，招募普通人 |
| 自愿角色者 | 主动接受自身剧本的人 |
| 星徒 | 基层信徒和执行者 |

## 代表人物

- 司星燃：真祭司。
- 祷星者：旧城钟楼事件执行者。
- 梁愿：自愿角色者代表。

## 长线变化

前期是神秘敌对组织。中期通过“自愿命运”获得大量普通人支持。后期在诸城会议中提出神话归位方案。终局后分裂：一部分成为保留信仰但尊重他人选择的团体，一部分成为极端残党。
