# 域管局组织结构

## 表层身份

政府下属异常事件管理部门，公开口径是城市灾害、精神异常、失踪事故和特殊治安事件处理单位。

## 真实职能

- 监控域潮汐。
- 封锁异常区域。
- 清理普通人异常记忆。
- 收容失控觉醒者。
- 与白塔、市集进行灰色交易以换取稳定。

## 组织层级

| 层级 | 功能 |
| --- | --- |
| 总署 | 制定跨城封锁政策，掌握最高风险档案 |
| 分局 | 负责具体城市的域事件处理 |
| 外勤队 | 进入现场封锁、撤离、清理 |
| 记忆处理组 | 修改普通人记录和社会口径 |
| 边界技术组 | 研究门、封印、铁域协议 |

## 代表人物

- 韩铎：总署代表，统一封锁协议推动者。
- 闻峻：旧城分局负责人，D 的旧上司。
- 纪槐：基层外勤，后期违抗命令打开撤离门。

## 长线变化

前期是追捕和封锁者。中期在七域失衡中证明封锁确实能减少伤亡。后期因牺牲区和跨城封锁引发道德危机。终局后重组为边界维护局，封锁权必须公开审查并允许申诉。
