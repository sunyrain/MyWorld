# 无名市集组织结构

## 本质

无名市集不是单纯地点，而是一套交易规则。只要有人相信命运、记忆、名字、关系和未来可以标价，市集就能出现。

## 交易品类

- 名字
- 记忆
- 梦
- 未来的一天
- 一段感情
- 某个人对你的恨
- 通往特定域的车票
- 原型共鸣份额
- 角色剧本

## 组织层级

| 层级 | 功能 |
| --- | --- |
| 市集主人 | 最高规则人格 |
| 拍卖师 | 主持高价值命运交易 |
| 债主 | 管理长期交易和关系债 |
| 掮客 | 为买卖双方牵线 |
| 账房 | 维护交易公平和账目 |
| 摊主 | 出售具体记忆、名字、梦和道具 |

## 代表人物

- 市集主人：最高规则人格。
- 红债主：E 的债主。
- 无槌：封闭拍卖拍卖师。
- 秦筹：市集自治城账房队长。
- 记忆铺老板：旧城和忆域线索提供者。

## 长线变化

前期是诡异交易地。中期变成命运经济系统，推动原型污染和未来拍卖。后期试图把改写成果商品化。终局后仍存在，但必须公开代价、设置撤销期，且不得永久买断反悔权。
