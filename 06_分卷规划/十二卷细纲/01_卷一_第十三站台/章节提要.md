# 卷一《第十三站台》章节提要

## 第 1-10 章：现实异常与无名信

| 章 | 本章事件 | 关系 / 设定推进 | 钩子 |
| ---: | --- | --- | --- |
| 1 | A 的日常生活中出现第一处异常：班级合照里多出一个没人认识的人。 | 建立普通都市日常，异常先以细节进入。 | 照片背面写着一个被划掉的名字。 |
| 2 | 城市新闻报道连续失踪案，第二天新闻链接消失。 | A 开始怀疑自己不是看错。 | 搜索记录里只剩一句“故事未完”。 |
| 3 | A 的同学忘记一个共同认识的人，只有 A 记得对方存在。 | 引出“被遗忘”作为第一卷恐惧。 | 同学通讯录里出现空白联系人。 |
| 4 | A 在地铁站看到不存在的站名一闪而过。 | 建立地铁系统与异常的关联。 | 站名显示为“第十三站台”。 |
| 5 | 第一名失踪者家属求助，A 发现对方也收到过无名信。 | 失踪案从新闻变成身边事件。 | 信上写着“你的故事还没有结束”。 |
| 6 | A 收到同样的信，信纸无法被撕毁或拍照保存。 | A 从观察者变成被选中对象。 | 信纸背面浮现末班车时间。 |
| 7 | A 试图报警，发现警员对“信”相关描述自动忽略。 | 表世界会自我修正异常。 | 警局墙上有域管局的隐秘标记。 |
| 8 | A 调查失踪者路线，发现他们都曾坐过同一班末班地铁。 | 主线目标明确：追查末班车。 | 地铁卡消费记录多出不存在的一站。 |
| 9 | A 夜里上车，车厢乘客逐渐变得沉默、模糊。 | 现实向域过渡。 | 车厢广播念出 A 的名字。 |
| 10 | 地铁停靠第十三站台，车门打开后现实声音全部消失。 | A 首次进入域。 | 站台标牌写着：请确认你的故事。 |

## 第 11-25 章：第十三站台与域规则入门

| 章 | 本章事件 | 关系 / 设定推进 | 钩子 |
| ---: | --- | --- | --- |
| 11 | A 发现站台上有许多等待离开的乘客，但没人记得自己要去哪里。 | 域会侵蚀人的目的和记忆。 | 乘客车票上写着不同死法。 |
| 12 | 站台广播要求乘客按“故事类型”排队。 | 初步出现“角色分配”概念。 | A 的队列栏为空白。 |
| 13 | A 遇到第一个被困者，对方因读完自己的车票故事而消失。 | 建立“读完结局会被域吞没”。 | A 的车票没有终点。 |
| 14 | A 尝试返回车厢，却发现车厢通向另一段站台。 | 域空间不遵循现实逻辑。 | 墙上地图显示雨巷、镜楼等站名。 |
| 15 | 站台检票员出现，要求 A 补全自己的故事。 | 域中存在规则执行者。 | 检票员无法念出 A 的全名。 |
| 16 | A 撕掉一张车票，短暂打断一名被困者的消失。 | A 的特殊能力首次无意识显露。 | 被救者立刻忘记 A 的脸。 |
| 17 | 站台开始追捕 A，广播称其为“未登记乘客”。 | A 成为域规则异常点。 | 远处有一盏暖光替他挡住追捕。 |
| 18 | A 跟随暖光逃入一条雨声不断的通道。 | 进入雨巷前奏，引出 B 的能力痕迹。 | 暖光尽头有人说“别读信”。 |
| 19 | A 在通道中看见失踪者残影，每个人都在重复一句遗憾。 | 域由情绪和记忆构成。 | 其中一个残影认识 A。 |
| 20 | A 发现第十三站台连接多个小型域，不是单一怪谈。 | 建立“站台是枢纽”。 | 站台深处有一扇被封住的门。 |
| 21 | 检票员再次出现，试图给 A 强行分配一个故事。 | 初步触碰“命运分配”主题。 | 被分配的故事标题是“无名者”。 |
| 22 | A 拒绝签名，检票口短暂失效。 | A 的能力与“拒绝被命名”相关。 | A 的手背出现淡淡墨痕。 |
| 23 | 站台坍缩，A 被迫选择进入雨巷或镜楼。 | 选择成为推进剧情的机制。 | 雨巷里传来求救声。 |
| 24 | A 进入雨巷，身后第十三站台恢复正常，像从未存在。 | 第一阶段结束，域不留下普通证据。 | 现实中的 A 被短暂遗忘一分钟。 |
| 25 | 雨巷第一盏路灯亮起，灯下放着写给 A 的信。 | 雨巷规则正式开启。 | 信封上写着“未寄出的告别”。 |

## 第 26-45 章：雨巷事件与献灯者 B

| 章 | 本章事件 | 关系 / 设定推进 | 钩子 |
| ---: | --- | --- | --- |
| 26 | A 在雨巷中遇到永远等待的人群。 | 雨巷主题确定：遗憾与执念。 | 每个人都在等一个不会回来的人。 |
| 27 | A 差点读完自己的信，被一束光打断。 | B 以救援者姿态登场。 | B 的手指冷得像冰。 |
| 28 | B 带 A 躲进屋檐，解释“读完信就会留下”。 | B 成为临时引导者。 | B 自己的信已经被雨水泡透。 |
| 29 | A 发现 B 正不断救人，却从不休息。 | B 的外在人设：温柔可靠。 | B 的影子比正常人淡。 |
| 30 | 雨巷居民请求 B 救回他们等待的人，B 无法拒绝。 | B 的伤口显露：被需要才有价值。 | A 看见 B 的灯火暗了一层。 |
| 31 | A 和 B 尝试救出一名失踪者，却发现对方自愿留下。 | 域不是单纯牢笼，也提供心理安慰。 | 失踪者说现实里没人等他。 |
| 32 | B 强行燃起灯火，为失踪者照出回路。 | 展示献灯者能力。 | B 的体温继续下降。 |
| 33 | A 质疑 B 为什么总是牺牲自己，B 回避。 | A 与 B 出现第一次价值碰撞。 | B 说“有人需要我就够了”。 |
| 34 | 雨巷规则升级，路灯下出现更多未寄出的信。 | 域会针对闯入者调整。 | 其中一封写着 B 的名字。 |
| 35 | B 的过去残影出现：他曾因没能救人而被留下罪恶感。 | B 的内在伤口具体化。 | 残影请求 B 再救一次。 |
| 36 | A 试图阻止 B 读信，B 反而认为自己必须承担。 | A 与 B 的信任出现裂缝。 | 雨水变成细小火星。 |
| 37 | B 读到一半，开始被雨巷写入“燃尽者”结局。 | 原型侵蚀初步显现。 | B 的影子几乎消失。 |
| 38 | A 撕开自己的空白信纸，短暂停止雨巷结局。 | A 能打断域的叙事结局。 | 雨第一次停了一秒。 |
| 39 | A 和 B 趁结局停顿救出部分被困者。 | 两人建立实际协作。 | 被救者出巷后忘记 B 的名字。 |
| 40 | B 因被遗忘而失控，仍想继续救所有人。 | B 的核心恐惧被触发。 | 灯火开始燃烧 B 自己。 |
| 41 | A 告诉 B：不被需要时也应该活着。 | A 第一次主动影响同伴选择。 | 雨巷深处传来鼓掌声。 |
| 42 | 雨巷显现域主残影：等待所有告别完成的邮差。 | 小型域也有核心意志。 | 邮差说 A 没有可投递地址。 |
| 43 | A 和 B 合力改写一封信的结尾，让等待者自己选择离开。 | 首次展示“新结局”雏形。 | 邮差记下 A 的空白名字。 |
| 44 | 雨巷打开出口，B 选择不再回头救最后一个幻影。 | B 获得第一步成长。 | 幻影微笑说“这次你可以走”。 |
| 45 | A、B 逃回第十三站台残留入口，发现镜楼站牌亮起。 | 雨巷事件结束，下一事件开启。 | A 的信纸多出镜面纹路。 |

## 第 46-65 章：镜楼事件与镜中骑士 C

| 47 | C 从镜楼中冲出，警告两人别碍事。 | C 以锋利、独行姿态登场。 | 她身后有一个更完美的倒影。 |

| 50 | B 看见一个不需要牺牲也被爱的自己，动摇。 | 雨巷成长被再次考验。 | 那个 B 邀请他留下。 |
| 51 | C 看见更优秀、更冷静、更被认可的自己。 | C 的核心伤口露出。 | 完美 C 称她为“失败版本”。 |
| 52 | 镜楼制造镜像战斗，C 展示镜像与反射能力。 | 展示 C 能力。 | 每次使用能力，C 的倒影更清晰。 |
| 53 | A 发现镜楼不是杀人，而是替换人。 | 小型域危险升级。 | 一名失踪者从镜中回到现实，但性格全变。 |
| 54 | C 独自深入，被完美镜像诱导承认自己不配存在。 | C 的原型侵蚀开始。 | 镜像拿走 C 的声音。 |
| 55 | A、B 决定救 C，尽管她此前拒绝他们。 | 团队关系第一次反向选择。 | B 的灯照不出 C 的影子。 |
| 56 | 镜楼让 A 看到所有人最终忘记他的未来。 | A 的恐惧被具体化。 | 未来中 B、C 都叫不出他的名字。 |
| 57 | A 差点接受镜楼给出的“有名字的人生”。 | A 也会被诱惑，不是天然正确。 | 镜像 A 已经拥有完整档案。 |
| 58 | B 用灯火唤回 A，但这次没有燃烧自己到极限。 | B 的成长被巩固。 | 灯光照出镜楼的核心楼层。 |
| 59 | 三人进入核心楼层，发现 C 的镜像准备替换现实中的她。 | 危机目标明确。 | 完美 C 已经能触碰现实物品。 |
| 60 | C 承认自己害怕不优秀就会被抛弃。 | C 第一次暴露内在伤口。 | 镜楼所有镜面同时开裂。 |
| 61 | A 尝试打断替换结局，但镜楼要求他写下自己的名字。 | A 的能力遇到代价。 | 写下名字会让他被镜楼收录。 |
| 62 | B、A 拖住镜像，C 自己选择不杀死完美镜像。 | 改写方式不是消灭可能性。 | 完美 C 第一次露出恐惧。 |
| 63 | C 接受不完美的自己，镜像失去替换资格。 | C 完成第一步成长。 | 镜楼留下“未完成的骑士”记录。 |
| 64 | 三人逃离镜楼，C 虽未正式加入，但开始同行。 | 三人小队成形。 | C 偷偷保存 A 的空白车票。 |
| 65 | 镜楼崩塌前映出无门医院，病房没有门。 | 指向卷二关键地点。 | 画面中有人喊 D 的代号。 |
