# 十二卷揭示节奏表

## 核心原则

核心世界观必须前移。前四卷完成基础揭示，后八卷写“知道真相之后怎么办”。

## 揭示节奏

| 卷 | 卷名 | 必须揭示 | 不能拖延到后面的问题 |
| --- | --- | --- | --- |
| 卷一 | 第十三站台 | 域存在，小型域有规则，能力有代价 | 不要只写怪谈氛围，要让读者看到规则 |
| 卷二 | 旧城钟楼 | 域与神话有关，名字可被夺走，主角团集结 | 不要把五人集结拖到中后期 |
| 卷三 | 白塔原型 | 原型是命运结构，能力来自伤口，侵蚀会把人写完 | 不要再用“神秘力量”含糊带过 |
| 卷四 | 域灾档案 | 远古域灾、第一批觉醒者、原型系统保护世界也囚禁人 | 不要把世界观底层真相压到终局 |
| 卷五 | 七域初潮 | 七大域是七种世界观，不只是地图 | 不要等到后期才让七域发挥作用 |
| 卷六 | 无名市集 | 原型可交易、污染、伪造，命运经济形成 | 不要只把市集写成支线集市 |
| 卷七 | 城市神话化 | 普通人也会被分配神话角色，城市可变巨型域 | 不要让主线只围绕觉醒者 |
| 卷八 | 诸城远征 | 其他城市有不同解决方案，世界规模打开 | 不要只在本城重复危机 |
| 卷九 | 阵营裂变 | 各阵营都掌握部分真相，也都会制造牢笼 | 不要写成简单正邪对抗 |
| 卷十 | 改写实验 | 神话可以被改写，但改写有失败和副作用 | 不要把“改写神话”留到最后才第一次尝试 |
| 卷十一 | 神话层战争 | 神话层是终局战场，不是首次讲解世界观 | 不要把这里写成大型说明书 |
| 卷十二 | 未完成神话 | 原型系统被改写，人获得多结局选择 | 不要摧毁所有神话，要改变神话与人的关系 |

## 读者体验目标

卷四结束时，读者应能说清：

- 域是什么。
- 原型是什么。
- 能力为什么有代价。
- 远古域灾大概发生过什么。
- 原型系统为什么既保护人也囚禁人。
- 主角团为什么要寻找第三条路。

卷五以后，读者期待的重点应从“真相是什么”转向：

- 真相会如何改变现实。
- 不同阵营会给出什么解决方案。
- 主角团能否真的改写神话。
- 改写失败会带来什么代价。
